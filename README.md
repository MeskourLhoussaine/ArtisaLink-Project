## :star2: About the Project

**ArtisaLink** is an innovative web platform designed to connect artisans and employers in Morocco.  
Artisans can post their services, and employers can browse profiles to find the right talent.  

The platform simplifies the matching process by providing:  
- Secure messaging system for communication between artisans and employers  
- Dashboard for tracking posts, interactions, and analytics  
- Multilingual interface supporting **Arabic, French, and English**  

ArtisaLink helps modernize the artisan sector in Morocco, making it easier for both service providers and employers to find each other.

---

## :computer: Technologies Used

- **Frontend:** Angular, Tailwind CSS  
- **Backend:** Java, Spring Boot, Spring Security, Spring Data JPA  
- **Database:** PostgreSQL  
- **Other Tools:** JWT authentication, RESTful APIs, Docker (optional for deployment)

---

## :rocket: Features

- User registration and profile management (artisans & employers)  
- Post and search services/offers  
- Internal messaging system  
- Analytics dashboard for tracking activity  
- Multi-language support (AR, FR, EN)
